# 📶 WiFi Password Security Scanner (Python)

A simple Python tool that analyzes the **strength of a WiFi password** and estimates how secure it is against hacking attempts.

This project demonstrates basic **cybersecurity principles** and helps users understand how to create strong passwords.

---

# 🚀 Features

✔ Checks WiFi password strength
✔ Detects numbers, uppercase letters, and special characters
✔ Provides security level (Very Weak → Strong)
✔ Estimates password crack time
✔ Beginner-friendly Python script

# 🧠 How It Works

The program evaluates the password based on:

* **Length Check** → Minimum 8 characters
* **Numbers** → Detects digits
* **Uppercase Letters** → Detects capital letters
* **Special Characters** → Detects symbols

Each successful condition increases the **security score**.
